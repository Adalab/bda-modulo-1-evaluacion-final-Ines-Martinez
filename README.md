# bda-modulo-1-evaluacion-final-Ines-Martinez
bda-modulo-1-evaluacion-final-Ines-Martinez created by GitHub Classroom
En este repositorio está incluido el ejercicio técnico para la evaluación del primer módulo, "Descubre el poder de Python" que imparte Adalab.
En este ejercicio técnico teníamos que crear La clase TiendaOnline. En mi caso, he pensado en una librería.
Para llevar a cabo el ejercicio técnico teníamos que crear un inventario (una lista de diccionarios) con los productos, su precio y la cantidad de cada uno, un diccionario de clientes y un registro de las ventas totales, con el fin de iterar por el inventario y comprobar la disponibilidad del producto, actualizar el stock, actualizar los datos, eliminar los artículos, calcular el valor del inventario y realizar una compra.
